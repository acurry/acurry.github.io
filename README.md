# andrew_d_curry_._com

## install hugo

```sh
$ brew install hugo
```

## run locally

```sh
$ hugo server -D
```

## build for prod

```sh
$ hugo --minify
```

## new blog post (draft)

```sh
$ hugo new posts/initial_post.md
```
